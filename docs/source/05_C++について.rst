名前空間スコープ
----------------------------------------------------------------

``namespace 名前 {}``で名前空間を定義できる．
``namespace``ブロックで括られたシンボルは，その名前空間に属する．

.. code-block:: cpp

    namespace foo {
        int x;
    }

    namespace bar {
        int x;
    }

    int main() {
        foo::x = 1;
        bar::x = 2;
    }
