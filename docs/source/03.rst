STM32 Nucleoについて
----------

PCからデータを受け取り，モータを制御する．

受け取るデータのフォーマットはTwist型である．

.. code-block:: 
    
    geometry_msgs/Twist
        float64 linear.x
        float64 linear.y
        float64 linear.z
        float64 angular.x
        float64 angular.y
        float64 angular.z

これをPC側で8bit x 4のデータに変換し，Nucleoに送信している．
- data[0] : 速度データの上位8bit
- data[1] : 速度データの下位8bit
- data[2] : 角速度データの上位8bit
- data[3] : 角速度データの下位8bit
